git流程：
主分支：main
日常对自己的分支 保持跟main 的更新一致：
在main 进行 git pull ， 然后在自己的分支做 git rebase

接到任务后：
--
create branch   （git checkout -b）

make your changes  (git add 文件名)

commit changes (git commit -m 'message')

push your branch (git push)

pull/merge request  (在网站上操作)
